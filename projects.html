Projects

Fortran Coarray – Open Source Contribution(work in-progress)
Created OpenSHMEM and UCCS communication layer support for Fortran Coarray library in OpenUH open source compiler. UCCS is a low-level communication substrate developed at Oak Ridge National Lab.

Performance Analysis for Intel Xeon Phi using hybrid models
Performed analysis on Intel Xeon Phi co-processors using Stampede Supercomputer in Texas Advanced Computing Center (TACC) to understand different performance tuning and optimizations for MPI, OpenMP and OpenSHMEM.

Compiler Intermediate Representation (IR) analysis
Developed various iterative data flow analysis algorithms for analyzing Open64 compiler Intermediate Representation. The IR for Open64 compiler is called WHIRL (Winning Hierarchical Intermediate Representation Language).

OpenMP and Image Processing
Parallelized the Optical Character Recognition application code for Vehicle Number plate analysis using OpenMP. Identified the hotspots in the OCR source code and parallelized those regions using OpenMP.

Software Design Pattern Implementation
Implementing a number of software design patterns like Facade, Abstract Factory and Observer to provide solutions to various software design problems using Test Driven Development in Java with JUnit and Mockito as testing framework.

Amazon AWS(work in-progress)
Created and hosted a web based movie data analysis application with automatic caching, load balancing and application abstraction in AWS cloud service using EC2, DynamoDB and Elastic MapReduce.

Bigdata Analytics
Various analysis and simulation were done on the US Flight data set using Hadoop modules like MapReduce, HDFS, YARN and HBase

Parallelizing Algorithms
Converted sequential versions of Image Smoothing, Jacobi, FFT and MD algorithms into parallel algorithms using OpenMP and OpenSHMEM.
